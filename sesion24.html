🚀 Caso Real - NASA Mars Climate Orbiter (1999)

🛰️

El Desastre

El Error

Confusión entre libras-fuerza y Newtons en los cálculos de navegación

El Costo

Pérdida total de $327 millones USD

9 años de trabajo perdidos

La Consecuencia

Orbiter se perdió al entrar en la atmósfera marciana

Datos del Proyecto:

Duración: 1998-1999
Equipo: NASA + Lockheed
Misión: Estudio clima Marte
Resultado: Pérdida total
🔍

Análisis RCA

Causa Inmediata

Software de navegación usó unidades imperiales (libras-fuerza) mientras el sistema esperaba métricas (Newtons)

Causas Contribuyentes

  • • Falta de estándares comunes entre equipos
  • • Comunicación deficiente NASA-Lockheed
  • • Ausencia de pruebas de integración
  • • No validación de unidades en interfaces

Causas Raíz

  • • Proceso de gestión de configuración inadecuado
  • • Falta de revisiones cruzadas de código crítico
  • • Presión de cronograma que omitió validaciones
  • • Cultura organizacional de "confianza ciega"

Acciones Correctivas

  • • ✅ Estándares obligatorios de unidades (SI)
  • • ✅ Pruebas de integración mandatorias
  • • ✅ Revisiones independientes de código crítico
  • • ✅ Validación automática de interfaces

💡 Lecciones Aprendidas

📏
