📊 Clasificación de Normas de Calidad

🌍

Normas Internacionales

ISO (International Organization for Standardization)

  • • ISO 9001: Sistemas de Gestión de Calidad
  • • ISO 27001: Seguridad de la Información
  • • ISO 14001: Gestión Ambiental
  • • ISO 20000: Gestión de Servicios TI

IEEE (Institute of Electrical and Electronics Engineers)

  • • IEEE 830: Especificación de Requisitos
  • • IEEE 1012: Verificación y Validación
  • • IEEE 828: Planes de Configuración
  • • IEEE 1028: Revisiones de Software
📈

Modelos de Madurez

CMMI (Capability Maturity Model Integration)

Nivel 1: Inicial (Caótico)
Nivel 2: Gestionado (Repetible)
Nivel 3: Definido (Estándar)
Nivel 4: Cuantitativamente Gestionado
Nivel 5: Optimizado (Mejora Continua)

Otros Modelos:

  • • SPICE (ISO 15504)
  • • TMM (Test Maturity Model)
  • • P-CMM (People CMM)
🇲🇽

Normas Nacionales

MoProSoft (México)

Modelo de Procesos para la Industria de Software

Alta Dirección: Gestión de Negocio
Gerencia: Gestión de Procesos, Proyectos y Recursos
Operación: Desarrollo y Mantenimiento de Software

Otras Normas Nacionales:

  • • COMPETISOFT (Iberoamérica)
  • • NMX-I-059-NYCE (México)
  • • Normas AENOR (España)
